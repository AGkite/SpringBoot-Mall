技术栈：Spring Boot + Mybatis Plus + Vue3 + Element Plus + tailwindcss

## 三、安装教程

- git pull 拉取此仓库

- 后端：springboot-mall

​			用IDEA打开springboot-mall目录。

- 前端：vue-mall
  1. 用VSCode打开vue-mall目录
  2. 运行`npm install`安装依赖。
  3. 运行项目`npm run dev`
